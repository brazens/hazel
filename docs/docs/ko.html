소개

Videt는 체코어로 '보다'라는 뜻입니다. videt의 의미로 videt은 시각적으로 보이는 요소를 간단하고 깔끔하게 구축하기 위한 웹 프레임워크입니다. videt은 다른 라이브러리와 통합하여 다양한 시각 시스템을 구축할 수 있으며 완전한 웹 지원을 제공합니다. videt은 누구나 무료로 사용할 수 있는 MIT 라이선스 오픈 소스 프로젝트입니다.

경고 :Videt은 IE9 이전 버전을 지원하지 않습니다.

설치하기

CSS를 로드 하려면 스타일시트 를 붙여 넣으세요.

<link rel="stylesheet" href="https://cdn.videt.xyz/css/0.1.2.min.css">

Image

It is designed to make it easy to change the transparency of an image. You can adjust the transparency with .img-opac-* , max value is 10 and min value is 1.

<img class="img-opac-1 center" style="width: 100px;" src="https://videt.xyz/images/full_logo.png">

The maximum value of 10 is the same as the original image.

Text

All text properties consist of .text-*

text-color-* : Change color. Read more
ext-left : Left-aligns text.
text-center : Center the text.
text-right : Right-aligns text.
text-upper : Change all text to uppercase.
text-lower : Change all letters to lowercase.
text-capit : Capitalizes only the beginning of a word.
text-line-top : A line is displayed above the text.
text-line-center : A line is displayed in the middle of the text.
text-line-bottom : A line is displayed below the text.
text-line-none : The line of text disappears. (You can delete a line by adding a hyperlink.)
text-overflow : If the text exceeds a certain part, it is displayed as ...
text-normal : Default text.
text-slant or text-italic : The text is slanted.
text-bold : Makes text bold.
text-bolder : Makes text bolder than the value inherited.
text-lighter : Makes it thinner than the value inherited from the text.
text-italic-bold or text-slant-bold : text bold and slanted.

Hover

hover properties are defined as hover-*

hover-bg-{color} : Change the background color with {color}.

hover-text-{color} : Change the text color with {color}.

hover-border-{color} : Change the border color with {color}.

position

position properties are defined as left , center , right

left : align position to the left.

center : align position to the center.

right : align position to the right.

Button

Use button styles that support different sizes and shapes. button properties are defined as btn-*

normal button

btn : Create a basic button.
btn-angled : Creates a button with right angles.
btn-round : Creates a button with slightly rounded corners.

button with border only

btn-border : Basic buttons have only borders.
btn-border-angled : Buttons with right angles have only borders.
btn-border-round : Buttons with slightly rounded corners have only a border.

normal button

button class="btn bg text-color-white".btn/button
button class="btn-angled bg text-color-white".btn-angled/button
button class="btn-round bg text-color-white".btn-round/button

button with border only

<button class="btn-border border">.btn-border</button>
<button class="btn-border-angled border">.btn-border-angled</button>
<button class="btn-border-round border">.btn-border-round</button>

Button tags : .btn can be used not only in button tag but also in a tag.

Navs

If you use videt navs basic components, you can create fast.
You can explore the basic configuration with the following example.

example
home Team forum Support
<div class="nav bg">
<a href="#" class="name text-color-white hover-text-white">example</a>
<div class="nav-right">
<a href="#home" class="text-color-white hover-text-white">home</a>
<a href="#team" class="text-color-white hover-text-white">Team</a>
<a href="#forum" class="text-color-white hover-text-white">forum</a>

</div>
</div>
