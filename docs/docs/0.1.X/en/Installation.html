Installation

Warning : Videt does not support IE9 or earlier.

Direct installation with link

Paste the stylesheet link to load the CSS

link rel="stylesheet" href="https://cdn.videt.xyz/css/0.1.2.min.css"

Occasionally older versions are deleted. To maintain the design of your website, keep it updated to the latest version.

Downloading with cdn takes some time. After the latest version is released, wait about 5 minutes and then you can use it as a cdn.
