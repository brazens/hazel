Image

transparency

It is designed to make it easy to change the transparency of an image. You can adjust the transparency with img-opac-* , max value is 10 and min value is 1.

img class="img-opac-5 center" style="width: 300px; height: 140px;" src="https://videt.xyz/images/full_logo.png"

The maximum value of 10 is the same as the original image.
